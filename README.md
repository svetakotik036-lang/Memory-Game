# Project Name
Memory Game

## Team Members
- Burlacu Svetlana
- Rusu Polina

## Project Idea
Memory Game este un joc de logică și memorie în care jucătorul trebuie să descopere perechi de cărți identice. Toate cărțile sunt așezate cu fața în jos, iar la fiecare mutare jucătorul întoarce două cărți. Dacă imaginile coincid, acestea rămân vizibile; dacă nu, sunt întoarse la loc.

## Motivation
Proiectul a fost ales pentru a dezvolta atenția, memoria și gândirea logică a utilizatorilor, dar și pentru a exersa programarea în Kotlin și lucrul în echipă folosind GitHub.

## Features
- întoarcerea cărților
- verificarea perechilor
- scor sau număr de mutări
- timer pentru joc
- interfață grafică simplă
- design simplu și intuitiv

## How to Run
1. Deschide proiectul în Android Studio
2. Rulează aplicația pe emulator sau telefon
3. Începe jocul
